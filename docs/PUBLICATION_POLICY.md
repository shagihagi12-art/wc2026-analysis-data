# Publication policy

## What to publish

- CSV data with timestamp and source-reference columns intact
- The workbook as a read-only analysis artifact
- Release notes that distinguish completed groups from groups still pending at the snapshot
- A short description of the Monte Carlo model’s limits

## What not to publish

- FIFA logos, event branding graphics, team crests, Match Centre screenshots, official photography, or video
- Claims that a retrospective A–F reconstruction was an original pre-kickoff public forecast
- A current-tense claim based on a dated snapshot without the snapshot time
- A “confirmed” qualifying claim when the relevant tie-break requires fair-play or ranking verification

## X / social-media rule

Every post must include:

1. the snapshot date and time;
2. a distinction between fact and estimate;
3. a link to the GitHub Release or repository;
4. no official logo, screenshot, or implied affiliation.
